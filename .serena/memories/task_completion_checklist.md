# Task Completion Checklist

When a task is completed, ensure:

1. **Code Quality**
   - [ ] Run `npm run lint` - fix all ESLint errors
   - [ ] TypeScript compilation successful (`npm run build`)
   - [ ] No console errors in browser

2. **Testing**
   - [ ] Test in development mode (`npm run dev`)
   - [ ] Test responsive design (mobile, tablet, desktop)
   - [ ] Test all interactive features (buttons, forms, links)
   - [ ] Verify WhatsApp integration works correctly

3. **Code Review**
   - [ ] Follow code style conventions
   - [ ] Proper TypeScript types
   - [ ] Accessible markup (semantic HTML, ARIA when needed)

4. **Documentation**
   - [ ] Update README if needed
   - [ ] Comment complex logic

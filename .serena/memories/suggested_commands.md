# Suggested Commands

## Development Commands
- `npm run dev` - Start development server with HMR
- `npm run build` - Build for production (TypeScript compilation + Vite build)
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint

## Package Management
- `npm install <package>` - Install dependencies
- `npm install -D <package>` - Install dev dependencies

## Windows PowerShell Utils
- `ls` or `Get-ChildItem` - List directory contents
- `cd <path>` - Change directory
- `Select-String -Pattern "<pattern>" -Path <file>` - Search in files (grep alternative)
- `git status` - Check git status
- `git add .` - Stage all changes
- `git commit -m "message"` - Commit changes

# ZonaEnglish Landing Page - Project Overview

## Purpose
Landing page untuk program ZonaEnglish School Ambassador 2025 - program untuk siswa SMP & SMA di Makassar yang ingin menjadi duta Bahasa Inggris di sekolah mereka.

## Tech Stack
- **Framework**: React 19.1.1 + TypeScript 5.9.3
- **Build Tool**: Vite 7.1.7
- **Styling**: Tailwind CSS (to be added)
- **Animations**: Framer Motion (to be added)
- **Icons**: Lucide React (to be added)
- **Dev Dependencies**: ESLint, TypeScript ESLint
